# v0.2.7

## New Features

### Mermaid Diagram Support
- **Full Mermaid integration** for creating diagrams in markdown and MDX
  - Flowcharts, sequence diagrams, class diagrams, and more
  - Automatic detection and rendering of mermaid code blocks
  - **Live theme switching** - diagrams re-render when toggling dark/light mode
  - Dark mode support - diagrams adapt to theme automatically
  - Error handling with helpful messages
  - Direct component usage in MDX files
  - Stores diagram code for re-rendering on theme change

### Diagram Types Supported
- Flowcharts and graphs
- Sequence diagrams
- Class diagrams
- State diagrams
- Git graphs
- Pie charts
- Entity relationship diagrams
- Journey diagrams
- And all other Mermaid diagram types

## Bug Fixes

### Search Improvements
- Improved SearchDialog with weighted ranking
- Results prioritized based on headlines (H1-H6) and content
- H1 matches weighted highest, content matches lowest
- Better relevance scoring

## Styling

### Search Visual Enhancements
- Search matches highlighted in `config.theme.colors.accent` color
- Consistent with app theme
- Bold weight for better visibility
- No more hardcoded yellow highlights

## Documentation

- Added comprehensive Mermaid diagrams guide
- Examples of all major diagram types
- Usage instructions for both MD and MDX
- Tips and best practices
- External resources and references
