# v0.2.9

## New Features

### Search and navigation UX

- Added keyboard navigation to the search dialog.
  - Arrow up/down moves between search results with wrap-around.
  - Enter activates the selected result.
  - Escape closes the dialog from the input.
- Highlighted search result is now visually indicated using the existing theme classes.

### Accessibility improvements

- Search dialog now exposes proper dialog semantics with `role="dialog"`, `aria-modal`, and `aria-labelledby`.
- Mobile floating toolbar menu is now exposed as a dialog with `aria-haspopup`, `aria-expanded`, and `aria-controls` on the trigger button.
- Sidebar navigation is now marked as a navigation landmark with an accessible label.

### Documentation

- Updated README with a short section describing search keyboard navigation and the floating toolbar behavior.
- Updated COMPONENTS.md with a section on navigation/search UX and how it integrates with theme classes.
