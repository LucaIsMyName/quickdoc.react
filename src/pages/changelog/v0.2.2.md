# V0.2.2

## Features

- **Document Footer**: Added comprehensive footer to all pages with meta information (created date, last updated, author, file name)
- **Theme-Aware Footer**: Footer respects theme configuration for colors, borders, and radius settings
- **Content Alignment**: Implemented `config.content.align` property to control content positioning (left, center, right)
- **Content Max-Width**: Added `config.content.maxWidth` support for any valid CSS max-width value
- **Enhanced Copy Buttons**: Copy buttons now respect theme border radius and size configuration
- **Improved Code Block Styling**: Code blocks and inline code now use consistent theme-aware backgrounds

## Bugfixes

- Fixed sidebar items when headline contained `inline code` - only multiline code blocks are now ignored from Table of Contents
- Fixed code block and inline code theme synchronization with website colors
- Fixed `config.content.align` property not working - content container now properly aligns based on configuration
- Fixed `config.content.maxWidth` not being applied to content container
- Fixed copy button positioning by adding `position: relative` to pre containers
- Fixed copy button styling to use theme colors and border configurations

## Documentation

- **Updated QuickDoc Documentation**: Comprehensive rewrite of all documentation files in `/src/pages/quickdoc/`
- **Enhanced Configuration Guide**: Added detailed examples for new color system, theme options, and all configuration properties
- **Updated File Structure**: Documented new components, utilities, and enhanced features
- **Improved Markdown Features**: Added MDX support documentation with interactive examples
- **Enhanced QuickStart Guide**: Complete rewrite with current features, proper file paths, and step-by-step setup
- **Updated Development Guide**: Added testing instructions, new dependencies, and enhanced workflow documentation
- **Integrated MDX Examples**: Moved MDX examples into markdown-guide for better organization
