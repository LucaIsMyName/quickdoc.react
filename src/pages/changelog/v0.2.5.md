# V0.2.5

## UI/UX Improvements

### Theme System Enhancements
- **Adaptive hover backgrounds** - Hover effects now use configured theme colors instead of hardcoded gray
- **Neutral navigation styling** - Sidebar and top navigation use neutral gray fonts for better readability
- **Proper background hierarchy** - Content area now has correct contrast with sidebar in both light and dark modes
- **Code block visual consistency** - Inline code and code blocks now use sidebar background color for clear distinction

### Color System Overhaul
- **New neutral color variables** - Added neutral color system for navigation elements
- **Improved color mapping** - Better separation between theme colors and neutral UI colors
- **Enhanced contrast** - Proper visual hierarchy between content and navigation areas

### Active State Improvements
- **Fixed active file highlighting** - Currently selected file now properly shows in accent color
- **Corrected hover behavior** - Hover backgrounds only appear on main navigation items, not subsections
- **Theme-aware interactions** - All interactive elements now respect theme configuration

## Code Block Styling

### Background Consistency
- **Unified code backgrounds** - Both inline code and fenced code blocks use consistent background colors
- **Theme integration** - Code block backgrounds match sidebar color for visual coherence
- **Border radius support** - Code blocks now respect theme border radius configuration
- **Border size support** - Optional borders on code blocks based on theme settings

### Visual Distinction
- **Clear content separation** - Code elements now stand out clearly from regular content
- **No border dependency** - Visual distinction achieved through background color contrast
- **Consistent theming** - All code elements follow the same visual pattern

## Component Fixes

### Sidebar Improvements
- **Background color correction** - Sidebar now uses proper theme-aware background colors
- **Text color optimization** - Neutral text colors for better readability across themes
- **Active state reliability** - Fixed active file detection and highlighting

### Navigation Enhancements
- **Tab navigation styling** - Top navigation tabs now use consistent neutral styling
- **Hover effect refinement** - Improved hover states for better user feedback
- **Border consistency** - Navigation borders now follow theme configuration

## Technical Improvements

### Color System Architecture
- **CSS variable expansion** - Added comprehensive neutral color variables
