# v0.2.6

## New Features

### MDX Component Library
- **Added 7 reusable MDX components** for enhanced documentation
  - `IFrame`: Embed YouTube videos, maps, and external content with responsive aspect ratios
  - `Callout`: Highlight info, warnings, success, and error messages with icons
  - `Tabs`: Organize related content in tabbed interfaces
  - `CodeBlock`: Enhanced code display with copy button and optional line numbers
  - `Steps`: Visual step-by-step guides with numbered circles
  - `Card`: Content containers with multiple variants (default, bordered, elevated)
  - `Badge`: Labels and tags with multiple colors and sizes

### Theme Integration
- **All components are fully theme-aware**
  - Automatically adapt to `config.theme.border.radius` settings
  - Respect `config.theme.border.size` configuration
  - Use theme color scheme (accent, light, dark)
  - Full dark mode support
- **New CSS variables**: `--theme-border-radius` and `--theme-border-size`
- **New CSS classes**: `theme-border-radius`, `theme-border-size`

### Vite Plugin Enhancement
- **Added HTTP middleware** to Vite plugin for proper MDX raw content loading
- Fixed path resolution for `/src/pages/` prefix in index.mdx files
- Improved raw content extraction for all MDX files
