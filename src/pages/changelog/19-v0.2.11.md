# V0.2.11

## Features

### Floating Toolbar Tooltips
- Added Base UI `Tooltip` components to the floating toolbar actions on desktop.
- Search, export, theme toggle and font-size toggle are now icon-only buttons with descriptive tooltips.
- Tooltips make the actions easier to discover without adding text labels to the toolbar itself.

## Bugfixes

### Floating Toolbar
- Fixed a TypeScript typing issue in `FloatingToolbar` for the font size toggle.
- The toggle now safely handles the current `fontSize` value and cycles between `small`, `medium`, and `large` without type errors.
- Updated navigation state tests to include the new `fontSize` field in `AppState` and to use the extended `getInitialState` signature.
