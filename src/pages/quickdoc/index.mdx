# QuickDoc

Complete developer documentation for customizing and deploying QuickDoc.

## Overview

QuickDoc is a documentation framework built with React, TypeScript, and Tailwind CSS. It automatically generates a navigable documentation site from your Markdown files with advanced features like cross-page navigation and search functionality.

### Key Features

- **Automatic Navigation**: Generates sidebar navigation from your Markdown headings
- **Hierarchical Numbering**: Optional numbered sidebar with hierarchical structure (1., 1.1., 1.2.)
- **Cross-Page Navigation**: Navigate directly between pages and sections with proper hash scrolling
- **Scroll to Top**: Automatically scrolls to top when navigating between files
- **Floating Toolbar**: Modern floating action toolbar with export, search, and theme controls
- **Keyboard Shortcuts**: Cmd+K / Ctrl+K to open search dialog
- **Enhanced Search**: Weighted search results prioritizing H1-H6 matches over content
- **Smart Search Highlighting**: Matches highlighted in accent color with bold weight
- **Export Functionality**: Export individual sections as PDF (print dialog) or Markdown files
- **Theme System**: Simplified color configuration with Tailwind color names (accent, light, dark)
- **Syntax Highlighting**: Code blocks with theme-aware backgrounds and copy-to-clipboard functionality
- **MDX Support**: Full MDX support with React components and interactive content
- **Mermaid Diagrams**: Full support for Mermaid diagrams with live theme switching
- **MDX Component Library**: 7+ reusable components (IFrame, Callout, Tabs, CodeBlock, Steps, Card, Badge)
- **Theme-Aware Components**: All components respect theme configuration (borders, radius, colors)
- **Error Boundaries**: Comprehensive error handling with graceful fallbacks
- **Responsive**: Mobile-first design with collapsible sidebar and overflow protection
- **SEO Optimized**: Dynamic meta tags and document titles with React Helmet
- **Type-Safe**: Full TypeScript support with comprehensive type definitions
- **Font System**: System fonts with Google Fonts override capability
- **Configurable Footer**: Optional sidebar footer with title, author, and year
- **Base UI Integration**: Modern accessible components with smooth animations
