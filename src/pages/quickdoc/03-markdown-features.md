# Markdown Features

## Code Blocks

QuickDoc supports syntax highlighting for code blocks:

```javascript
function example() {
  console.log("Hello, QuickDoc!");
}
```

### Copy to Clipboard
All code blocks include a copy button for easy copying.

## Links and Navigation

### Internal Links
Link to other pages and sections:
```markdown
[Configuration](/quickdoc/configuration)
[Cross-page anchor](/quickdoc/configuration#fonts)
```

### External Links
Standard markdown links work as expected:
```markdown
[GitHub](https://github.com)
```
