import React from 'react';

# Interactive Counter

Here's a simple React component embedded in the documentation:

### Component Overview

This section explains the counter component.

export const Counter = () => {
  const [count, setCount] = React.useState(0);
  
  return (
    <div style={{ 
      padding: '20px', 
      border: '2px solid #3b82f6', 
      borderRadius: '8px', 
      textAlign: 'center',
      margin: '20px 0',
      backgroundColor: '#f8fafc'
    }}>
      <h3 style={{ color: '#1e40af', marginBottom: '15px' }}>Count: {count}</h3>
      <button 
        onClick={() => setCount(count + 1)}
        style={{
          backgroundColor: '#3b82f6',
          color: 'white',
          border: 'none',
          padding: '10px 20px',
          borderRadius: '5px',
          marginRight: '10px',
          cursor: 'pointer'
        }}
      >
        Increment
      </button>
      <button 
        onClick={() => setCount(0)}
        style={{
          backgroundColor: '#ef4444',
          color: 'white',
          border: 'none',
          padding: '10px 20px',
          borderRadius: '5px',
          cursor: 'pointer'
        }}
      >
        Reset
      </button>
    </div>
  );
};

<Counter />

### Usage Notes

This counter component maintains its own state and demonstrates how React components work within MDX content.

### Technical Details

The component uses React hooks for state management.
