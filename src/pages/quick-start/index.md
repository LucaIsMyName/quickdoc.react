# Quick Start

Get up and running with QuickDoc in minutes.

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/yourusername/quickdoc.react.git
cd quickdoc.react
npm install
```

## Add Your Documentation

Create or edit Markdown files in the `public/pages/` directory:

```bash
# Example: Create a new documentation file
echo "# My Documentation" > public/pages/My-Docs.md
```

## Start Development Server

Run the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

Create an optimized production build:

```bash
npm run build
```

The build output will be in the `dist/` directory, ready to deploy to any static hosting service.

## What's Next?

- **Customize**: Edit `src/config/app.config.ts` to personalize your documentation site
- **Learn More**: Check out the **QuickDoc** page for detailed configuration options
- **Write Docs**: See the **Markdown Guide** for tips on writing great documentation
